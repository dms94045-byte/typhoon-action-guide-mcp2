# 태풍 대응 행동 가이드 MCP PRD

## 1. 제품 개요
**태풍 대응 행동 가이드 MCP**는 기상청 태풍 예보를 바탕으로, 국민이 태풍 상황에서 가장 궁금해하는 **상륙·통과 시간과 위치**를 생활 시간대 기준으로 재구성하고, **지금 당장 해야 할 행동**을 빠르게 안내하는 대화형 도구이다.

본 MCP는 태풍 예보를 생성하거나 예측하지 않으며, **가장 최신의 공식 예보 정보를 행동 판단에 적합한 형태로 해석·전달**하는 것을 목적으로 한다.

---

## 2. 기획 배경 및 문제 정의
### 2.1 문제 상황
- 태풍 발생 시 정보는 충분하지만, 국민은 다음 질문에 즉시 답을 얻기 어렵다.
  - “몇 시에 우리 지역이 가장 위험한가?”
  - “지금 나가도 되는가?”
  - “오늘 밤에 무엇을 준비해야 하는가?”
- 기상청 예보는 정확하지만, 전문 용어와 시각 중심 정보로 인해 **긴급 상황에서 즉각적인 행동 판단**으로 이어지기 어렵다.

### 2.2 해결 방향
- 태풍 예보를 **국민의 하루 시간 흐름(밤/새벽/출근 시간 등)**으로 재해석
- 지역·환경별로 **가장 위험한 시간대와 행동 지침을 압축 제공**
- 검색 없이도 대화만으로 핵심 정보를 획득 가능하도록 설계

---

## 3. 타깃 사용자
- 태풍 영향권에 있는 일반 국민
- 관광객 및 지역 거주자
- 재난 상황에서 빠른 판단이 필요한 사용자

---

## 4. 핵심 가치 제안 (Value Proposition)
- **언제 위험한지 바로 알 수 있다**
- **우리 지역 기준으로 설명해준다**
- **지금 해야 할 행동만 알려준다**

---

## 5. 주요 기능 정의

### 5.1 태풍 상륙·통과 시점 요약 기능
- 최신 태풍 예보를 기반으로
- 상륙/통과 가능 시점을 **시간대 중심(오늘 밤, 내일 새벽, 출근 시간 등)**으로 설명
- 단정적 표현이 아닌 가능성 기반 안내

### 5.2 사용자 지역 기반 맞춤 안내 기능
- 사용자가 입력한 지역(예: 제주, 부산, 남해안, 내륙 등)을
- 영향 유형(해안/내륙/산간/저지대)으로 해석
- 해당 유형에 맞는 위험 요소와 행동 지침 제공

### 5.3 가장 위험한 시간대 강조 기능
- 태풍 영향 전체를 나열하지 않고
- 사용자에게 **가장 주의해야 할 핵심 시간대 1개**를 명확히 제시

### 5.4 상황별 행동 지침 자동 생성 기능
- 입력 정보(지역, 시간대, 태풍 단계)를 종합하여
  - 반드시 해야 할 행동
  - 하면 안 되는 행동
  - 한 줄 요약
  형태로 제공

### 5.5 정보 미입력 시 필수 질문 유도 기능
- 사용자가 상황을 입력하지 않은 경우
- 행동 판단에 꼭 필요한 최소 정보만 단계적으로 질문
- 선택형 질문 중심으로 긴급 상황에서도 부담 최소화

### 5.6 기준 시점 명시 기능
- 모든 응답에
  - “○월 ○일 ○시 기준 예보 반영” 문구 포함
- 실시간에 근접한 정보 제공임을 명확히 전달

### 5.7 예보 불확실성 설명 기능 (보조)
- 예보 변경에 대한 사용자 질문에 대해
- 예보의 특성과 변동 가능성을 짧고 이해하기 쉽게 설명

---

## 6. 사용자 흐름 (User Flow)

1. 사용자가 MCP 호출
2. 상황 입력 여부 판단
   - 입력 있음 → 즉시 맞춤 안내
   - 입력 없음 → 최소 질문 제시
3. 최신 기준 예보 반영
4. 시간·지역 기반 행동 가이드 제공
5. 한 줄 요약으로 행동 판단 마무리

---

## 7. 응답 기본 구조

- 기준 정보 (예보 기준 시각)
- 태풍 이동 및 통과 예상 요약
- 지금 반드시 해야 할 행동 (3개 내외)
- 하면 안 되는 행동 (2개 내외)
- 한 줄 행동 요약

---

## 8. 실시간성 설계 원칙

- 태풍 예보를 생성하거나 예측하지 않음
- 공식 예보 갱신 주기를 반영하여 최신 정보 사용
- 실서비스 시 API 연계, 프로토타입에서는 구조만 제시

---

## 9. 기술 및 구현 범위 (MVP)

- 인증 없음
- 위치 정보 자동 수집 없음 (사용자 입력 기반)
- 외부 API 연계는 선택 사항
- 텍스트 기반 대화형 인터페이스

---

## 10. 확장 가능성

- 지역 세분화 (권역 → 시·군 단위)
- 취약 환경 선택 옵션 추가
- 시각적 타임라인 제공
- 다국어 지원

---

## 11. 한 문장 요약

**태풍 대응 행동 가이드 MCP는 태풍 예보를 ‘언제·어디가 위험한지’로 재구성해, 국민이 즉시 행동을 결정하도록 돕는 공공형 대화 도구이다.**
